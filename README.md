# 图片压缩

> 纯前端图片压缩工具，质量滑块调节，实时预览对比，原始/压缩大小显示

## 功能概览

- **选择/拖拽图片加载，支持 PNG/JPG/WebP 格式**
- **质量滑块 1-100 精细调节，实时显示压缩后预览**
- **原始大小 vs 压缩大小对比显示 + 压缩率百分比**
- **下载压缩后图片，支持保留原格式或转换格式**
- **纯 Canvas 前端处理，不上传服务器，隐私保护**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
